
# Rust Websocket Server
This is a websocket server to handle data communication between clients in a Unity scenario
where position, rotation and slices are sent.

# Project structure

    .
    ├── Cargo.lock
    ├── Cargo.toml
    ├── Makefile
    ├── README.md
    └── src
        ├── config
        │   └── mod.rs
        ├── handlers
        │   └── mod.rs
        └── main.rs


# Run the project

```bash
make run
```
